iOS证书获取

Issho Lin 3/10/2024 app开发

# Bundle ID

在Dcloud开发者中心，找到对应的应用，查看各平台信息列表，与包名保持一致

# 证书

登录苹果开发者中心 (opens new window)，找到证书选项，点击进去申请

# identifiers

点击identifiers - 点击新增

选择第一个选项 App IDs - 点击继续

选择App

输入标识符的描述和Bundle ID（Bundle ID的格式最好是 com.domianname.appname）

记得在capabilities里选中自己需要的功能，例如需要推送功能，就选择push notifications

所有都填好了，就回到顶部点击继续

再次确认信息，确认后点击registe

这时候回到identifiers列表中就可以看到你注册的id了

# Certificates（私钥证书p2）

点击新增

选择对应的是开发证书，还是生成正式证书

这时就进入填CSR文件的页面

# 生成CSR文件

在Mac电脑上操作，找到 【钥匙串访问】 应用，点击进去，选择 证书助力-从证书颁发机构请求证书

填写好对应的信息

这样就得到了CSR

# 获取cer文件

将获取到csr上传

点击继续后就可以下载cer文件了

# 生成对应的私钥文件 p12

拿到cer文件在mac中点击打开 进入钥匙串访问

在【我的证书】选项中，选择你对应的cer文件 然后 右键选择导出

点击导出

要记住这里输入的密码，这个密码就是证书私钥密码

这里输入mac电脑锁屏密码

点击允许后就获得了对应的p12 私钥文件了

# Profiles（证书profile文件）

点击新增

这里的appid就是前面创建的id

这样就获取成功了

打包时把对应的信息填写即可
